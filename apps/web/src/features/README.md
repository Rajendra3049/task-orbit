# Features Directory

This directory contains all domain-first modules for TaskOrbit.

Each feature is isolated and owns its components, hooks, services, and types where relevant.

Current feature modules:

- `auth`
- `dashboard`
- `tasks`
- `projects`
- `habits`
- `goals`
- `calendar`
- `analytics`
- `ai`
- `workspaces`
- `settings`
